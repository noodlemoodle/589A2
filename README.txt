TO COMPILE AND RUN IN TERMINAL:

>>make run
Follow prompts to enter parameters for the hypocycloid.
Note that program will exit if parameters are not positive real numbers.

Key Commands:

    Animation Controls:

     LEFT SHIFT - toggle animation

    Object Controls (Only in static mode):

     KEYPAD_7 - Rotate CCW in x
     KEYPAD_9 - Rotate CW in x
     KEYPAD_4 - Rotate CCW in y
     KEYPAD_6 - Rotate CW in y
     KEYPAD_1 - Rotate CCW in z
     KEYPAD_3 - Rotate CW in z
     KEYPAD_2 - Rotate CW in all axis
     KEYPAD_5 - Rotate CCW in all axis
     KEYPAD_ADD - Scale up
     KEYPAD_SUBTRACT - Scale down
     SPACE - Reset to default state

    Window Controls:

     ESC - Close window
     ENTER - Reset input parameter
